You are running the peg-risk safety gate for the asset identified in the argument (mint address or token name). Follow `assess.md` exactly. Do not skip steps or invent numbers.

## Step 1: Classify the Asset

Determine the asset class from the mint address or token name. Reference `peg-states.md` for the 8 classes (`stable_fiat`, `stable_cdp`, `stable_rwa`, `stable_dn`, `stable_fx`, `lst`, `synth_lev`, `stable_yield`).

Questions to classify:
- Fiat-backed stable pegged to $1? → `stable_fiat`
- CDP stable with collateral ratio? → `stable_cdp`
- Stake-pool LST? → `lst`
- Yield-bearing stable with per-share NAV? → `stable_yield`
- Non-USD fiat peg? → `stable_fx`
- Leveraged synthetic? → `synth_lev`

If the class cannot be determined from the mint or name alone, use Helius DAS (`getAsset` with `showFungible: true`) to inspect the token's metadata and issuance mechanism. If still unknown, emit UNKNOWN and stop — do not guess.

## Step 2: Pull Intrinsic and Market

Follow the per-class recipe from `computing-spread.md`.

Mandatory checkpoints before proceeding:
- Is the intrinsic source within its staleness bound? If not → UNKNOWN.
- Is the intrinsic non-zero? If zero or null → UNKNOWN (FM-5 guard).
- Is the market quote non-null? If null (thin mint, no route) → UNKNOWN.

For Tier-2 assets (hyUSD CR via Hylo on-chain accounts, xSOL NAV via Hylo program): if the adapter is not implemented, print `"intrinsic source not bundled — see computing-spread.md"` and use UNKNOWN for that path.

## Step 3: Compute Discount and Run Guards (in order)

```
discount = 1 - market / intrinsic
```

Run in this exact sequence:

1. **Plausibility** (FM-4): if `|discount| >= 1.0`, the market quote is degenerate. Skip this tick → UNKNOWN. Do not use this sample in any EWMA.
2. **EWMA** (if prior state available): `smoothed = 0.4 * discount + 0.6 * prev_smoothed`. First observation: use raw discount.
3. **NAV-sanity** (FM-5): if direction-sensitive class (`lst`, `stable_yield`) AND `discount < 0` AND `|discount| * 10000 > 1000` → UNKNOWN (broken intrinsic).
4. **Direction-aware classification**: if direction-sensitive AND `discount < 0` → PEGGED. Otherwise, classify by `|discount| * 10000` against per-class thresholds from `peg-states.md`.
   - For `stable_cdp`: run both spread path and CR path; report the more severe.
5. **Hysteresis** (if prior state available): a less-strict new state requires the smoothed discount to fall below `threshold * (1 - 0.25)` (spread) or above `threshold * (1 + 0.02)` (CR) before confirming the downgrade. Escalation is never slowed.

## Refusals

Emit a blocking UNKNOWN (no RISK-PARAMS block) when:
- Asset class cannot be determined
- Intrinsic is unavailable, zero, or a Tier-2 class without an implemented adapter
- Market quote is null
- Both sides are stale beyond their bounds

Do not fabricate numbers. Do not substitute an undocumented price source without naming it. An assessment that produces a confident number from missing data is exactly FM-5.
